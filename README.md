# PSY-60-FORZA-SHADER-LOADER

Welcome to the **PSY-60-FORZA-SHADER-LOADER** repository by PSYCHOWHO / SIFAT!

This CLI tool helps you easily install ReShade and apply custom shaders/presets for Forza.

## Features

- **Automated ReShade Installation:** If you haven't installed ReShade, the CLI will download the latest setup for you automatically.
- **Easy Shader Deployment:** Copies all `reshade-shaders` files and `.ini` presets directly into your Forza game folder.
- **Beautiful CLI Interface:** A colourful terminal UI featuring custom ASCII art.
- **Developer Links:** Automatically directs you to the developer's Instagram after a successful installation!

## How to Use and Share

There are two ways you can share and use this program:

### 1. As a Standalone Executable (Recommended for sharing)
I have compiled the code into a standalone `installer.exe` file. This means anyone can run it without needing Python installed!
To share it:
1. Select **`installer.exe`** and the **`reshade-shaders`** folder.
2. Put them in a ZIP file (Right-click -> Compress to ZIP file).
3. Send that ZIP file to your friends or upload it! 
4. They just need to extract it and double-click `installer.exe` to run the program.

### 2. Using Python (For Developers)
If you want to run the raw code directly:
1. Open your terminal (Command Prompt or PowerShell).
2. Navigate to this repository's folder:
   ```cmd
   cd path\to\PSY-60-FORZA-SHADER-LOADER
   ```
3. Run the following command:
   ```cmd
   python installer.py
   ```
4. Follow the on-screen instructions!
